# Diagram Prompts

These prompts were used to call ChatGPT image generation for Japanese hand-drawn style README diagrams. The repository also includes reproducible local PNGs generated by `scripts/generate_handdrawn_diagrams.py`.

## 01 Overview

Create a Japanese hand-drawn style workflow diagram for a GitHub README. Theme: from public health data to SCI submission-ready package. Beige paper texture, soft blue green orange accents, hand-drawn arrows, cute but professional, flat sketch style. Include concise English labels: Data, Research Question, Analysis, Manuscript, Review, Submission Package. No logos, no people, no tiny unreadable text. 16:9 landscape.

## 02 Checkpoint Loop

Create a Japanese hand-drawn style process loop diagram for a GitHub README. Theme: checkpoint-first research harness loop. Beige paper texture, soft blue green orange accents, hand-drawn circular arrows. Include concise English labels only: Plan, Produce Artifact, Validate, Human Confirm, Next Stage. Show a small stop sign near Human Confirm to indicate no auto-continue. Professional tutorial style, readable text, 16:9 landscape.

## 03 Harness Architecture

Create a Japanese hand-drawn style architecture diagram for a GitHub README. Theme: Academic Research Harness architecture. Beige paper texture, soft blue green orange accents, hand-drawn boxes and arrows. Include concise English labels: Skill Router, Workflow, Stage Contract, State JSON, Validator, Artifacts, Human Confirmation. Show them connected as an engineering system, not a marketing poster. No logos. Readable text. 16:9 landscape.

## 04 NHANES Case Path

Create a Japanese hand-drawn style case-study path diagram for a GitHub README. Theme: NHANES 2017-2018 small data pack becomes tables, figures, manuscript, and Word submission package. Beige paper texture, soft blue green orange accents, hand-drawn icons for XPT files, R analysis, tables, figures, manuscript DOCX. Include concise English labels: NHANES Small Pack, Survey Analysis, Table 1/2, Figures, Manuscript Draft, DOCX Package. No logos. Readable text. 16:9 landscape.
