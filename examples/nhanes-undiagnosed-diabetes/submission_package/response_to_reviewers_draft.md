# Response to Reviewers Draft

Note: This response letter is based on the simulated S8 major-revision review and should be adapted if an actual journal decision letter is received.

## General Response

We thank the editor and reviewers for their constructive comments. We revised the manuscript to improve statistical transparency, insert complete tables and figures, clarify the fasting-glucose sensitivity analysis, strengthen interpretation boundaries, and add reporting details relevant to a cross-sectional NHANES study.

## Major Revision Items

### Comment 1: Insert actual tables and figures rather than placeholders.

Response: We inserted Table 1 and Table 2 into the manuscript and generated Figure 1 and Figure 2. Supplementary tables for BMI sensitivity, categorical screening model, fasting sensitivity, and covariate missingness were also prepared.

### Comment 2: Clarify analytic sample sizes and missingness.

Response: We added explicit sample-size reconciliation. The primary analytic sample included 4,004 participants. The main waist-circumference model included 3,621 participants, and BMI/categorical sensitivity models included 3,707 participants. Key covariate missingness is now reported.

### Comment 3: Recalculate the fasting-glucose sensitivity analysis using fasting subsample weights.

Response: We recalculated the fasting-subsample sensitivity analysis using `WTSAF2YR`. The fasting-weighted prevalence increased from 1.84% using HbA1c alone to 4.06% using HbA1c or fasting plasma glucose.

### Comment 4: Report design constraints and avoid overparameterized interpretation.

Response: We added the final survey design degrees of freedom (15) and clarified that model parsimony was prioritized because this was a single-cycle NHANES analysis.

### Comment 5: Strengthen equity-oriented interpretation and avoid biological essentialism.

Response: We revised the race/ethnicity discussion to frame NHANES race/ethnicity variables as social categories shaped by access, structural factors, and measurement context. We added targeted references on racial/ethnic differences among adults with undiagnosed diabetes and diabetes screening.

### Comment 6: Add data availability and ethics statements.

Response: We added Data Availability and Ethics Statement sections noting the public, de-identified nature of NHANES data and NCHS ethics oversight.
