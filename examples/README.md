# Examples

This directory contains one complete teaching example and two scaffolded public
database templates.

| Directory | Status | Purpose |
|---|---|---|
| `nhanes-undiagnosed-diabetes/` | Complete S0-S9 teaching example | Demonstrates data, analysis, tables, figures, manuscript, revision trail, and submission package |
| `charls-aging-template/` | Scaffold only | Initializes CHARLS project governance after registered data access is confirmed |
| `gbd-burden-template/` | Scaffold only | Initializes GBD query/export governance after a Results Tool export or approved API route is available |

Do not treat scaffold directories as completed analyses. CHARLS and GBD claims
should remain draft-only until approved data/export files, S3 analysis outputs,
and S7 integrity checks exist.
