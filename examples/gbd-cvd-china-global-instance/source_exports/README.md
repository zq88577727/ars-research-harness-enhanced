# Source exports

This directory may contain the local real GBD Results Tool export used by the
CVD China/global manuscript candidate:

`gbd_cvd_china_global_1990_2023.csv`

`gbd_cvd_citation.txt` stores the citation text bundled with the export.

The CSV must match `../gbd_cvd_query_manifest.csv`. The source CSV is ignored by
default and should remain local unless IHME/GHDx reuse terms are explicitly
confirmed for public repository redistribution. The citation text can remain in
the repository because it is needed to audit the manuscript citation.
