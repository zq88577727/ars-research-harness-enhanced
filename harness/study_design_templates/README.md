# Study Design Templates

These templates make the harness portable beyond the NHANES cross-sectional
example. Each template declares the reporting guideline and the minimum stage
contracts that must be satisfied before drafting or finalization.

Included templates:

- STROBE observational cross-sectional studies
- CONSORT randomized trials
- PRISMA systematic reviews and meta-analyses
- TRIPOD prediction model studies
- RECORD real-world evidence studies
- STARD diagnostic accuracy studies
