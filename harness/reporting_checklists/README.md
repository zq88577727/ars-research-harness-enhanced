# Reporting Checklists

This directory stores structured reporting-guideline audits. Markdown checklist
exports are useful for reading, but JSON checklists are easier to validate,
diff, and reuse across manuscripts.

Recommended status values:

- `addressed`: the manuscript or submission package contains the needed information.
- `partial`: the item is started but still scientifically incomplete.
- `not-addressed`: the item has no adequate evidence.
- `not-applicable`: the item does not apply to the study design.
- `author-dependent`: the harness cannot truthfully complete the field without
  author input, such as funding source, conflicts of interest, author
  contributions, or acknowledgements.

Author-dependent items should not be counted as scientific validation failures,
but they must be resolved before real journal submission.
